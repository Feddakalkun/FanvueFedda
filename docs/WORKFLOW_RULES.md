# Workflow Rules

## Non-Negotiable Logging Rule

After every fix or code change, update `H:\Fanvue_use_this\MEMORY_LOG.md` immediately.

## Definition of "Fix/Change"

- New file
- File edit
- Refactor
- Config/script change
- Bug fix

## Minimum Per-Change Log Fields

- Date
- Time (local)
- Change title
- Files touched
- Why the change was made
- Verification method/result
